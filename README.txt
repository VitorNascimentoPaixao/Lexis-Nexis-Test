Versão utilizada: Python 3.9
Autor: Vitor Nascimento Paixão
Bibliotecas utilizadas: -math(padrão)

Abra o programa MAIN, rode o código.
	1. Digite o número de linhas da matriz desejada. Pressione ENTER.
	2. Digite o número de colunas da matriz desejada. Pressione ENTER.
	3. Em seguida, serão exibidas as coordenas da matriz na tela, de 	maneira sequencial. Digite o número desejado para cada coordenada. 	Pressione sempre ENTER após digitar o número escolhido.
	4. Quando forem digitados os valores de todas as coordenadas da 	matriz ela aparecerá completa na tela.
	5. O algoritmo construído irá zerar as linhas e colunas onde forem 	encontrado algum número 0
	6. A matriz resultante será exibida logo abaixo.
	7. O programa encerra.
